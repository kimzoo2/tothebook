th:text

1. if-unless

example:
<span class="output" th:if="${user.isTemporaryPassword}" >비밀번호를 변경해주세요.< /span >
<span class="output" th:unless="${user.isTemporaryPassword}" > 로그인 페이지로 < /span >
result : 비밀번호를 변경해주세요. 로그인 페이지로

2. swtich

example:
< span th:switch="${user.isTemporaryPassword}">
 < span th:case=true>임시비밀번호 < /span>
 < span th:case=false>원래비밀번호 < /span>
</span>
result : 임시비밀번호 원래비밀번호
