th:text

1. 변수 값 가져오기

example: <span th:text="${user.email}"> < /span>
result : spring@tothebook.co.kr

2. 숫자 연산

example: <span th:text="|1 + 1 = ${1 + 1}|"> < /span>
result : 1 + 1 = 3

3. 비교 연산

example: <span th:text="${user.joinDate}?:'이력이 없습니다.'"> < /span>
result : 2022년 5월 3일

4. 날짜 포맷터

example: <span th:text="${#temporals.format(user.joinDate, 'dd-MM-yyyy HH:mm')}"> < /span>
result : 2022년 5월 3일
